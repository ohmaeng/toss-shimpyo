# 쉼표 정류장 (전국판)

버스정류장의 **그늘·의자·조명·도착안내기** 유무를 3상태(있음/없음/미확인)로 알려주는 고령자용 시민 웹앱과, 유개식 승강장(쉘터) 설치 후보를 조건 필터로 뽑는 행정 대시보드.

**전국 확장 버전** — 춘천시 해커톤 본선작(`Chuncheon_shimpyo`)의 앱 디자인과 코드를 이어받아, 전국 버스정류장 데이터·실시간 도착정보(국토부 TAGO)로 확장하는 것을 목표로 합니다.

- 시민앱: `/` — 켜자마자 지도 + 내 주변 정류장 휴식시설
- 행정 대시보드: `/admin` — 설치 후보 TOP N + CSV 내보내기

## 데이터 방향 (전국 확장)

- **정적 정류장**: `data/국토교통부_정류장 현황_20250613.csv` — 전국 정류장 좌표·정류소ID(약 12MB). 확장의 기반 데이터.
- **실시간 도착정보**: 국토부 TAGO 버스도착정보 API(전국 대부분 시·군). `app/.env` 의 `VITE_TAGO_KEY` 필요, 없으면 폴백 표시.
- **데모 데이터**: `app/public/data/stops.json` 은 현재 춘천 1890개 정류장(앱 구동/시연용). 전국판에서는 뷰포트/지역 분할 로딩으로 대체 예정.
- 파이프라인(`pipeline/`)과 기획 문서(`docs/`)는 춘천판 기준 — **전국 확장 참고용**으로 보존.

## 1. 앱 개발 서버

```bash
cd app
npm install
npm run dev
```

브라우저에서 http://localhost:5173 접속. 시민앱은 `/`, 대시보드는 `/admin`.

## 2. 정적 빌드 (PWA 오프라인)

```bash
cd app
npm run build
npm run preview
```

## 3. 테스트

```bash
cd app && npm test        # 프론트 (vitest)
cd pipeline && python -m pytest   # 파이프라인(참고)
```

## 설정 (선택)

- 실시간 버스 도착정보를 쓰려면 `app/.env.example` 을 `app/.env` 로 복사하고 `VITE_TAGO_KEY` 에 공공데이터포털 TAGO 키를 넣으세요. 없어도 폴백으로 동작합니다. **키는 커밋 금지.**

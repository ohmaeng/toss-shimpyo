## 🛑 즉시 중단 조건 (Halt Conditions — 2026-07-16 Codex 최종 승인 시 부과)

아래 상황이 발생하면 **작업을 멈추고 팀(총괄)에 보고한다. 테스트를 수정해 통과시키는 방식의 우회 금지.**

1. **unknown이 설치 검토 후보(InstallRow)에 들어오는 순간 중단.** 설치 트랙의 대상은 로드뷰/현장 조사로 no가 확정된 정류장뿐이다. 관련 테스트(A4 Step 1a)가 거슬리면 테스트가 아니라 접근을 고칠 것.
2. **검수되지 않은 로드뷰 결과(AI 초안)가 no를 생성하는 순간 중단.** no 생성 경로는 "사람이 확정한 조사 CSV" 단 하나다(A5). 초안 CSV를 `data/`에 넣거나 파이프라인에 물리는 코드는 작성 금지.
3. **데이터 품질 검증(A1)·사용자 테스트(A10)가 완료되기 전에 P2 기능(B3 카운터·브리핑)에 착수하는 순간 중단.** 우선순위 역전 금지.

**실행 지침:** 기능 개수를 늘리는 제안은 기본 거절. 대상 가능성은 ①조사 CSV 투입 전후 수치 라이브 데모(설계 §6 서사) ②데이터 검증 리포트(A1) ③사용자 테스트(A10)의 **안정적 완성**에서 나온다 — 이 세 가지가 다른 모든 것보다 우선한다.

## Global Constraints

- 정직성 규칙 유지 + v3.1 강화: **`확인된 없음(no)`과 `미확인(unknown)`을 같은 변수·같은 순위 효과로 합치는 코드 경로 금지**(테스트로 강제). 미확인은 조사 트랙에만, no는 설치 트랙에만.
- 합성 지수 5조건(산식 공개·실측 병기·출처 배지·B2C 숫자 비표시·변수 분리). B2C 정렬은 **"확인된 시설 존재 우선"**(미확인을 없음으로 단정하지 않음 — "확실성 선호"·"편안함 점수" 표현 금지), B2G 산식에 comfortScore 사용 금지.
- 표현 금지어: "현장 확인", "수혜", "일평균"(근거 확보 전), "그늘 4곳뿐"(→ "확인 4곳·미확인 1,886곳"), 조건 없는 "오프라인 동작"(→ "최초 1회 접속 후").
- 접근성: 타이핑 없이 완결 **그리고** 키보드만으로 조작 가능. 폰트 ≥18px, 터치 ≥48px, 아이콘+한글, WCAG AA.
- 키 보안: 클라이언트 번들 LLM 키 금지, 공개 배포 무키 빌드. stops.json/stop.ts 계약 불변(변경 필요시 총괄 회부).
- 커밋: 태스크마다 `git add <명시 경로>`. 메시지 끝 `Co-Authored-By: Claude Opus 4.8 (1M context) <noreply@anthropic.com>`.

**공통 타입 (`app/src/types/priority.ts`):**

```ts
export type PresetKey = "heat" | "senior" | "usage";           // 폭염/고령자/이용량
export interface SurveyWeights { demand: number; unknown: number; poi: number; }
export const PRESETS: Record<PresetKey, {label: string; w: SurveyWeights; rationale: string}>;
export interface SurveyRow {              // 1단계: 조사 우선순위 (수요 실측 보유 정류장만)
  stop: Stop; rank: number; score: number;             // score는 B2G 보조 표시용
  demandMidday: number; demandQ: number;                // 실측·분위수
  unknownCount: number; unknownRate: number;            // 미확인 시설 수/4
  poi: number | null;                                   // 인접도. 미확보 시 null(항 비활성)
}
export interface NoDemandCandidate { stop: Stop; unknownCount: number; } // 수요 미확인 그룹(순위 없음, 표 하단 별도)
export type FacilityKind = "seat" | "shade" | "light";
export interface InstallRow {             // 2단계: 시설별 설치 검토 (해당 시설 no 확인 정류장만)
  stop: Stop; facility: FacilityKind; rank: number;
  demandMidday: number | null; poi: number | null;
  surveySource: "roadview"; capturedAt?: string;
}
```

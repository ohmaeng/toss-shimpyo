## 현재 구현과 목표 기획의 차이

- 현재 앱의 즐겨찾기·정류장별 QR·목적지 경로·정적 관리자 순위는 과거 구현이다.
- 목표 기획의 공통 QR, 시민 음성 불편 구조화, 서버 집계, AI 근거 카드, 사람 확정 후 시민 환류는 별도 구현이 필요하다.
- 발표에서는 `현재 구현`, `본선 신규 구현`, `향후 협약 필요`를 구분한다.
